# World of Warcraft Game Data APIs Source: https://community.developer.battle.net/documentation/world-of-warcraft/game-data-apis Encompass both static and dynamic game data. For namespace details see: guides/namespaces ## Host Names Non-China: {region}.api.blizzard.com (us, eu, kr, tw) China: gateway.battlenet.com.cn ## Namespaces Required on every request. Each endpoint specifies the correct namespace. Submit as header (Battlenet-Namespace: {namespace}) or query (?namespace={namespace}). ## Search Certain data types support search. See guides/search. ## Endpoint Listing (partial — see source URL for full list) ### Achievement API GET /data/wow/achievement/index GET /data/wow/achievement/{achievementId} GET /data/wow/media/achievement/{achievementId} GET /data/wow/achievement-category/index GET /data/wow/achievement-category/{achievementCategoryId} ### Auction House API GET /data/wow/connected-realm/{connectedRealmId}/auctions GET /data/wow/auctions/commodities ### Azerite Essence API GET /data/wow/azerite-essence/index GET /data/wow/azerite-essence/{azeriteEssenceId} GET /data/wow/search/azerite-essence GET /data/wow/media/azerite-essence/{azeriteEssenceId} NOTE: The full page lists many more API groups alphabetically including: Connected Realm, Covenant, Creature, Guild Crest, Item, Journal, Media Search, Modified Crafting, Mount, Mythic Keystone, Mythic Raid, Pet, Playable Class, Playable Race, Playable Specialization, Power Type, Profession, PvP Season, PvP Tier, Quest, Realm, Region, Reputations, Spell, Talent, Tech Talent, Title, Toy, WoW Token, and more.
