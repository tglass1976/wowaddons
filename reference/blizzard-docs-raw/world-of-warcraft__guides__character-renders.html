# World of Warcraft Character Renders Source: https://community.developer.battle.net/documentation/world-of-warcraft/guides/character-renders Character render images are generated when a character logs out of World of Warcraft. ## Character Media Summary Endpoint /profile/wow/character/{realm-slug}/{character-name}/character-media Namespace: profile-{region} See profile-apis for full endpoint documentation. ## Fallback Images The character render CDN can serve fallback images for avatars that don't exist. To request a fallback, add the alt query parameter: | Parameter | Format | Example | |-----------|--------|---------| | ?alt | /shadow/avatar/{race-id}-{gender-id}.jpg | /shadow/avatar/2-1.jpg | Full example: {avatar_url}?alt=/shadow/avatar/{race-id}-{gender-id}.jpg ### Gender IDs | Gender | ID | |--------|----| | Male | 0 | | Female | 1 | ### Race IDs Fetch from: GET /data/wow/playable-race/index?namespace=static-us ### Sample Request https://render-us.worldofwarcraft.com/character/tichondrius/00/000000000-avatar.jpg?alt=/shadow/avatar/2-1.jpg
