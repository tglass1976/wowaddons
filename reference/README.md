# Reference Folder

This folder stores Blizzard World of Warcraft Developer Portal material that is useful while building addons and related tools.

## Important scope note

The Battle.net Developer Portal documents **web APIs** (HTTP + OAuth), not the in-game WoW addon Lua API directly.

- Use this for external tools, sync services, importers, or companion apps.
- Use in-game Lua APIs for addon runtime behavior inside WoW.

## What is included

- Local raw HTML snapshots of key WoW docs pages.
- A distilled implementation reference for auth, namespaces, localization, and region behavior.
- Source-page index so you can refresh or verify details.
- An AI-oriented reference pack with structured YAML/JSON guidance.
