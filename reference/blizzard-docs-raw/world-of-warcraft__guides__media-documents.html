# World of Warcraft Media Documents Source: https://community.developer.battle.net/documentation/world-of-warcraft/guides/media-documents Media documents help with the discoverability of web assets related to API resources. They can link to: web-rendered images, player character renders, in-game assets (icons). ## Pattern A resource document contains a media key linking to a generic media document. The media document contains references to web resources for the parent. ### Sample Item Document (excerpt) { "media": { "key": { "href": "https://us.api.blizzard.com/data/wow/media/item/19019?namespace=static-us" }, "id": 19019 } } ### Sample Media Document { "_links": { "self": { "href": "https://us.api.blizzard.com/data/wow/media/item/19019?namespace=static-us" } }, "assets": [ { "key": "icon", "value": "https://render-us.worldofwarcraft.com/icons/56/inv_sword_39.jpg" } ], "id": 19019 } IMPORTANT: Cache copies of asset resources on your own servers or storage. Do NOT link directly to Blizzard CDN resources in production.
