# World of Warcraft Search Source: https://community.developer.battle.net/documentation/world-of-warcraft/guides/search The WoW Game Data APIs support search. Returns paginated results, maximum 1000 entries. Currently supported document types: realm, connected-realm. ## Querying Data Base URL pattern: /data/wow/search/{documentType}?namespace={namespace} To filter on a field: /data/wow/search/{documentType}?namespace={namespace}&{field}={value} For nested fields (dot notation): /data/wow/search/connected-realm?namespace=dynamic-us&realms.timezone=America/New_York Multiple fields (implicit AND): /data/wow/search/connected-realm?namespace=dynamic-us&has_queue=true&realms.timezone=America/New_York ## Special Parameters | Parameter | Example | Description | |-----------|---------|-------------| | _tag / _tags | _tag=item | Media doc type filter (item, spell, creature-display, etc.) | | _page | _page=2 | Page of results (default: 1) | | _pageSize | _pageSize=500 | Results per page (default: 100, min: 1, max: 1000) | | orderby | orderby=field1:desc,field2:asc | Comma-separated sort fields with :asc/:desc | | namespace | namespace=dynamic-us | Required namespace | ## Operations | Operation | Syntax | Example | |-----------|--------|---------| | AND (implicit) | Multiple params | str=5&dex=10 | | OR | \|\| between values | type=man\|\|bear\|\|pig | | NOT | != | race!=orc | | NOT + OR | Combined | race!=orc\|\|human | | RANGE inclusive | [min,max] | str=[2,99] | | RANGE exclusive | (min,max) | str=(2,99) | | MIN | [min,] | str=[41,] | | MAX | [,max] | str=[,77] |
